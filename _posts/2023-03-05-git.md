```bash
-git

git init : 새로운 git 저장소를 생성할 때 사용하는 명령어

git status : 현재 파일의 상태를 확인할 때 사용하는 명령어

git add . : 수정한 내용을 저장하는 공간에 추가하는 명령어

git config user.name "SangHu"
git config user.email "bagsanghu90@gmail.com"

git commit -m”메세지” : 프로그램 변형으로 새로운 변화가 생겼을 때 이를 “ ” 안에 적어 내용을 기록하는 명령어

git remote add origin [https://github.com/hoohoo1220/github.io.git](https://github.com/hoohoo1220/github.io.git)

git push origin master : 웹사이트에 변경된 내용을 추가할 때 사용하는 명령어

git clone : 원격으로 저장하는 곳을 바꿈
```
